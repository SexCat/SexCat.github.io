Java基础（一）

内容来自廖雪峰老师官网学习笔记

Java版本

Java SE：Standard Edition

Java EE：Enterprise Edition

Java Me：Micro Edition

SE翻译为标准版，包含标准的Jvm和标准库，EE为企业版，在SE上增加了大量的API和库，SE和EE使用的虚拟机相同。

名词解释

JDK：Java development kit

JRE：Java Runtime Environment

JRE就是运行java字节码的虚拟机，如果只有java源码，要编译成Java字节码，就需要JDK，因为JDK除了包含JRE，还提供了编译器，调试器等开发工具。

JDK安装以及环境变量

Windows：

Java_HOME只想JDK安装目录

Mac：

.bash_profile 或.zprofile

把Java_HOME目录的Bin添加到系统环境变量PATH中

Path=%Java_HOME%\bin

Classpath

.,%Java_HOME%\bin\dt.jar;%Java_HOME%\lib\tools.jar

JDK下bin目录文件

Java：这个可执行程序其实就是JVM，运行Java程序，就是启动JVM，然后让JVM执行指定的编译后的代码；

Javac：这是java的编译器，将java源码编译成字节码class文件。

Jar：用于把一组class文件打包成jar文件。

Javadoc：用于从java源码中提取注释并生成文档。

Jdb：Java调试器，用于开发阶段的运行调试。

Hello World

Java程序开发三步骤：写代码、编译代码、运行代码

编译java源文件

javac HelloWorld.java

编译后发现当前目录下多了一个HelloWorld.class文件，该文件是java可运行的文件，称为字节码文件。

运行Java程序

java HelloWorld

入门程序说明

编译是将我们的java源文件翻译成jvm认识的class文件。

关于main方法

称为主方法，写法是固定的格式，main是程序的入口点。

注释Comment

单行注释：//开头，换行结尾

多行注释：/*开头，以*/结束

关键词

解释：在程序中，Java已经定义好的单词，具有特殊含义

在之前的案例中出现的关键词有，public、void、main、class等

标识符

解释：是指在程序中，我们自己定义的内容，比如类的名字、方法的名字、变量的名字，都是标识符。

常量

解释：在Java程序中固定不变的数据

整数常量、小数常量、字符常量、字符串常量、布尔常量、空常量

变量

解释：常量是固定不变的量，在程序中可以变化的量叫变量。

Java中要求一个变量每次只能保存一个数据，必须要明确保存的数据类型。

Java数据类型分类：

基本数据类型：整数、浮点数、字符、布尔

引用数据类型：类、数组、接口

四类八种基本数据类型：

数据类型 关键字 内存占用 取值范围
字节型 byte 1字节
整型 int（默认） 4字节
字符型 char 2字节

Java中默认类型：整数int，浮点是double

变量的定义：数据类型、变量名，变量值

数据类型 变量名=数据值

练习Demo

long类型：建议数据后加L表示

float类型：建议数据后加F表示

注意事项

变量名称：同一个大括号范围内，变量的名字不可以相同

变量赋值：定义的变量，不赋值不能使用
